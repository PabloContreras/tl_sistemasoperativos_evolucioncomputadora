Evolución histórica de las computadoras y sistemas operativos

Por:
Albor Meza Daniela
Contreras Gutiérrez Pablo Einar
Reyes Fernández Ricardo

500AC

El ábaco fue el primer dispositivo que ayudó al ser humano en la resolución de los problemas aritméticos.

1642

Se inventó la primera máquina para sumar que también se usaba para restar.

1834

Se inventó la primera computadora digital, era programada por una serie de tarjetas perforadas que contenían en datos las instrucciones.

1930

Se inventó una máquina analógica que utilizaba circuitos eléctricos.

1940

Se accedía directamente a la consola donde se actúa sobre una serie de microinterruptores. No existían los programadores y los SO para ese entonces.

1945

Se inventó la ENIAC, primera computadora electrónica, se construyó en Pensilvania para calcular tablas de artillería.

1950

Los SO hacen una aparición discreta y bastante simple tales como: el Monitor residente, pensamiento por lotes y almacenamiento temporal.

1951

1951-1958

La UNIVAC fue diseñada por Eckerd. Utilizaba tubos al vacío. PRIMERA GENERACIÓN DE LAS COMPUTADORAS.

1960

Nuevos métodos para gestionar los buffers y los dispositivos de entrada/salida.

1964

1959-1964

SEGUNDA GENERACIÓN, eran rápidas y pequeñas, nuevos lenguajes llamados lenguajes de alto nivel.

1971

1964-1971

TERCERA GENERACIÓN, más rápidas y más pequeñas.

1970

Se trata de archivos de texto sin formato, guardados con la extensión BAT que contienen un conjunto de comandos MS-DOS.

1976

AT&T desarrolló un protocolo de comunicación con el SO UNIX.


1980

1971-1980

CUARTA GENERACIÓN, el procesador de la computadora se le denominó como microprocesador.

1980

MS-DOS (siglas de Microsoft Disk Operating System, Sistema Operativo de Microsoft) es un SO para computadoras basados en x86.

1982

1982-1984

Lanzamiento oficial del ordenador Macintosh con el SO MacOs.

1990

1980-1990

QUINTA GENERACIÓN, con capacidad de proceso paralelo.


1990



SEXTA GENERACIÓN, publicaron Windows.

1990

Linux es un núcleo libre de SO basado en Unix. Es uno de los principales ejemplos de software libre. Windows 95 fue un SO con interfaz gráfica de usuario híbrido de entre 16 y 32 bits.

2000

Windows XP: ambiente gráfico más agradable que el de sus predecesores. Secuencias más rápidas de inicio y de hibernación. Android y Iphone OS 1.0. Android es un sistema operativo móvil basado en Linux.

2009

Windows 7: esta versión está diseñada para uso en PC, incluyendo equipos de escritorio en hogares y oficinas, equipos portátiles, tablet PC, netbooks y equipos media center.

2010

Windows Phone es un SO móvil desarrollado por Microsoft, como sucesor de la plataforma Windows Mobile.

2011



Mac OS X 10.6. Snow Leopard es la séptima versión del SO de Apple Mac OS X.

2012

Windows 8 producido por Microsoft para su uso en computadoras personales, y de negocios, computadoras portátiles, netbooks, tabletas, servidores y centros multimedia.

2013



Google presenta una nueva versión de su SO de Android.


2015

Windows 10 es el último SO desarrollado por Microsoft como parte de la familia de SO Windows NT.
